Core Utilities of PvZ: Replanter

A tool for replacing assets inside the bundles of Plants Vs. Zombies Replanted

This tool can automate the:
  - Decompression of bundles
  - Unpacking of bundles
  - Writting .emip files to the asset files

Still work in progress, of course.
It's very hacky, so make sure to back up your files before using this tool
